
Ejercicio 3: Sistema de Puntuación para un Juego

Descripción: Crear un sistema de puntuación simple para un juego donde los jugadores pueden ganar y perder puntos.

Requerimientos:

Primer ciclo: Implementar la funcionalidad para sumar puntos a un jugador.
Segundo ciclo: Implementar la funcionalidad para restar puntos a un jugador.
Tercer ciclo: Implementar la funcionalidad para consultar la puntuación actual de un jugador.
Cuarto ciclo: Implementar una funcionalidad para resetear los puntos del jugador a cero.
Clase principal: GestorPuntuacion

Método sumarPuntos(String jugador, int puntos)
Método restarPuntos(String jugador, int puntos)
Método obtenerPuntos(String jugador)
Método reiniciarPuntos(String jugador)
Cada estudiante puede realizar un ciclo TDD para cada funcionalidad de puntuación, permitiendo a todos contribuir al sistema completo.
